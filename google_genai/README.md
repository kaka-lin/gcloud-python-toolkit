# Google Gemini API

## Install

```bash
$ pip3 install google-genai
```

## Prerequirest

1. Gemini API Key: You can [create](https://aistudio.google.com/app/apikey) your API key using Google AI Studio with a single click. And then past it to your `.env` file

## Run

```bash
$ python3 chat.py
```
